I. Problem Statement

An integer constant with initialization in Pascal is a named variable whose value is set at declaration and does not change during program execution. The integer constant must have the data type `integer` and be initialized with a value at the time of declaration.

Syntax: const VariableName: integer = Number; where
  • const - a required keyword indicating that this is a constant.
  • VariableName - the name of the variable (constant), which must be unique within its scope.
  • integer – an integer data type.
  • Number – the value assigned to the constant at initialization.
Examples:
  • const abc: integer = 5;
  • const d: integer = 44;
