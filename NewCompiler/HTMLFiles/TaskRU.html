I. Постановка задачи

Целочисленная константа с инициализацией на языке Pascal — это именованная переменная, значение которой устанавливается при объявлении и не изменяется на протяжении выполнения программы. Целочисленная константа должна иметь тип данных integer и определяться с указанием значения сразу при объявлении.

Формат записи: const ИмяПеременной: integer = Число; где
  • const - обязательное ключевое слово, обозначающее, что это константа.
  • ИмяПеременной - имя переменной(константы), которое должно быть уникальным в пределах области видимости.
  • integer – целочисленный тип данных.
  • Число – значение, которое присваивается константе при инициализации.
Примеры:
  • const abc: integer = 5;
  • const d: integer = 44;
